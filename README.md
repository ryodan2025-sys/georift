# 🚀 STELLAR STRIKE

Mobil ve masaüstü tarayıcıda çalışan uzay gemisi shooter oyunu.

## 🎮 Oyna

👉 **[stellar-strike.github.io](https://KULLANICI_ADIN.github.io/stellar-strike)**

## 🕹️ Kontroller

| Platform | Hareket | Ateş |
|----------|---------|------|
| 📱 Mobil | Sol joystick | ⚡ Sağ buton |
| 🖥️ PC | Ok tuşları / WASD | Boşluk |

## 🌊 Oyun Özellikleri

- Sonsuz dalgalar — her dalga zorlaşır
- 3 farklı düşman tipi (UFO, Dart, Tank)
- Her 3. dalgada **BOSS** geliyor
- Güçlendiriciler: 🛡 Kalkan · ⚡ Hızlı ateş · ❤ Can

## 🚀 GitHub Pages'e Yayınlama
